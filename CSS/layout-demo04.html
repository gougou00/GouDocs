人类再败！柯洁对决人工智能首战告负 三番战开门黑

北京时间5月23日，现世界排名第一的柯洁与人工智能围棋程序AlphaGo展开三番棋决战的第一战。最终柯洁执黑负1/4子。双方的第二局比赛将于25日10：30继续进行。

去年三月，李世石面对AlphaGo，最终被1-4击败。今年初的时候，AlphaGo化身master，在各大网站对弈平台60-0横扫中日韩各路高手。

近一个半月，柯洁正式比赛8胜4负战绩并不算突出，但在围甲联赛的一些比赛中，都能看出他在演练和AlphaGo对阵的招数。发布会上，柯洁没有了一年前的“豪言”，表示：“我将尽全力争胜，一决胜负，抱有必胜的信念和必死的决心，不轻易言败。”

早上9时，赛前开幕式正式举行，柯洁、古力、陈耀烨、周睿羊、时越、唐韦星、芈昱廷、连笑等中国棋手到场。和去年对阵李世石时不同，在时间分配上，当时李世石对AlphaGo的用时是每方2小时，1分钟读秒5次，而此次则是每方用时3小时，1分钟读秒5次，增加用时适当地“照顾”了人类棋手。

10点半，比赛正式开始，经过猜先柯洁执黑先行。柯洁前四手棋下了两个三三，而星位直接点三三也是AlphaGo的标志性招法，柯洁选择“以彼之道，还施彼身”。之后，柯洁与AlphaGo交战点在棋盘右下和左上方。

双方通过厮杀，柯洁负1/4子。白棋在面对黑棋点三三和左上的攻防中表现出了强大实力的一角，将黑棋KO以后就又回到稳妥取胜的道路中。

柯洁：AlphaGo真是太厉害了 输得没脾气

柯洁表示，布局也有备战和研究，开始也有针对性下法，下三三后又点角，这都是AlphaGo喜欢下的，但没想到AlphaGo应对非常好，很多招法我都没见过，包括那步刺（白50），让我很震惊。首先断完以后，他变得很厚，还有很多劫材，可以说是一石二鸟。真是太厉害了，我输得没什么脾气。

此外，柯洁还表示AlphaGo确实下得太精彩，很多地方都值得我们去学习、探讨，思想和棋的理念，改变我们对棋的最初的看法，今天我也一直贯彻先捞后洗的战术，先把实地钞票捞到手，但在角部还是被他掏到实地，打破了我的战术，一下子进入他的步调。感觉AlphaGo和去年判若两人，当时觉得他的棋很接近人，现在感觉越来越像围棋上帝。我希望尽全力去拼每一盘棋。很感谢有AlphaGo这样的对手，感谢DeepMind团队给我机会去下这三盘棋，也希望通过这次比赛让大家了解围棋这个好项目，给大家带来快乐。
